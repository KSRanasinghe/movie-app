# 🎬 MovieApp

A simple Movie Search App built to practice **React** and **Tailwind CSS**.

## Key Features
- **OMDb API Integration:** Fetching real-time movie data.
- **Tailwind CSS Layout:** Responsive design using a mobile-first approach.
- **Local Storage CRUD:** Saving and managing a watchlist without a backend.
- **State Management:** Practicing React hooks (useState, useEffect).

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **Icons:** Tabler Icons (`@tabler/icons-react`)
- **API:** OMDb API

## How to Run
1. `npm install`
2. `npm run dev`
